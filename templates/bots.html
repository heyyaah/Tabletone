Bot API

Используйте HTTP API для управления ботом из своего приложения.

POST /bot/sendMessage

Отправить сообщение пользователю

{
  "token": "ВАШ_ТОКЕН",
  "chat_id": 123,
  "text": "Привет!"
}
GET /bot/getMe?token=ВАШ_ТОКЕН

Получить информацию о боте

POST /bot/setWebhook

Установить webhook для получения сообщений

{
  "token": "ВАШ_ТОКЕН",
  "url": "https://yourserver.com/webhook"
}
Webhook получает POST-запрос с JSON при каждом новом сообщении боту. Бот может ответить через /bot/sendMessage.
